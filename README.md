# FS Application Index

React 製作的應用程式與網站入口頁。

## 開發

```bash
npm install
npm run dev
```

預設本機網址：

```text
http://127.0.0.1:5173/
```

## APP 下載檔放置位置

請把安裝檔放在：

```text
public/downloads/
```

例如：

```text
public/downloads/your-app.exe
```

React 裡面的下載連結使用：

```text
/downloads/your-app.exe
```

部署後，如果網站是 `https://example.com`，實際下載網址會是：

```text
https://example.com/downloads/your-app.exe
```
